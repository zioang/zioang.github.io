图的定义

  图(Graph) 是由一个 非空的顶点集合V(Verex) 和一个描述 顶点之间多对多关系的边（或弧）集合E(Edge) 组成的一种数据结构，它可以形式化地表示为：G＝（V，E）， 其中:
$${V={v|v \in 某个数据对象集}}，$$
$$ E={（v_i，v_j）|v_i，v_j \in V}，$$
$$ 或 E={<v_i，v_j> | v_i，v_j \in V且<v_i,v_j>表示从v_i到v_j的一条单向通路}$$

  通常 V(G)： 表示图G的顶点集。E(G)： 表示图G的边集。 E(G) 可以是空集，若E(G) 为空，则图G只有顶点、没有边。综上所述，我们可以发现，其实 线性表和树 也都是图的一种。

  以杭州地铁规划图为例，我们可以将每个 站点 都看作 图的顶点，然后可以把 站与站之间的轨道 看作 图的边。最后，我们将所有站点归为集合V，而把所有轨道归为集合E，这样便组成了一个图。


基本术语

I、无向图 : 若每条边均 没有方向 ，则称G为无向图。无向图中的边均是 顶点的无序对，无序对通常用圆括号表示。


II、有向图 : 若每条边均 有方向 ，则称G为有向图。有向边又称为弧，弧的始点称为弧尾，弧的终点称为弧头 。


III、网络 : 在图的 每条边上 附上相关的 数值，这种与图的边相关的数值叫 权 。权可以表示 两个顶点之间的距离、耗费等具有某种意义的数 。若将图的每条边都赋上一个权，则称这种 带权图 为 网络 。 其中，每条边既可以是有向的，也可以是无向的。


IV、完全图 : 每对顶点之间 都恰巧 连有一条边 的简单图我们将其称为 完全图 ，而完全图也可以分为 有向完全图 和 无向完全图。

  无向完全图 : 其每对不同的顶点之间都恰巧连 有一条边相连 。 由其性质可得，无向完全图边(e)与顶点(n)的关系公式: e = n(n-1)/2 。


  有向完全图 : 其每对不同的顶点通过 一对唯一的边缘（每个方向一个） 连接。由其性质可得，有向完全图边(e)与顶点(n)的关系公式: e = n(n-1) 。


V、顶点的度 : 与 该顶点相关联的边的数目 ，顶点v的度记为 D(v) 。

VI、顶点的入度 : 有向图中，以 顶点v为终点的边的数目 ，顶点v的入度记为 ID(v) 。

VII、顶点的出度 : 有向图中，以 顶点v为始点的边的数目 。顶点v的出度记为 OD(v)。

VIII、有向图中，顶点各度之间的关系 : D(v) = ID(v) + OD(v)

IX、顶点数(n)、边数(e)和度数D(v)之间的关系 : $$ e = \frac{1}{2} * \sum_{i=1}^n * D(v_i)$$

X、子图 : 若给定两个图 Gi 和 Gj ，其中 Gi =（ Vi，Ei ），Gj=（ Vj，Ej ），若满足 Vi ⊆ Vj，Ei ⊆ Ej ，则称 Gu 是 Gj 的子图 。

XI、路径 : 一个长度为 k 的路径是一个 非空的顶点和边的交错序列 v0 e0、v1 e1 … vk-1 ek-1 ，使得对于所有 i < k 均有 ei = vi * vi+1 。特别的，当 v0 = vk 时，称这个路径是闭的(closed) ，当路径中的顶点互不相同，得到 G的一条路 。

XII、路径长度 : 该路径上 边或弧的数目 。

XIII.简单路径 : 如果一条路径上 除了起点v0和终点k相同外，其余顶点均不相同 ，则称此路径为一条 简单路径。起点和终点相同(v0 = vk)的简单路径称为 简单回路或简单环 。
